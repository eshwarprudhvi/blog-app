# 📝 blog-app

A full-stack blog application built with React, Node.js/Express, and MongoDB.

🌐 **Live Demo:** # 📝 blog-app

A full-stack blog application built with React, Node.js/Express, and MongoDB.

🌐 **Live Demo:** https://blog-app-prudhvi.netlify.app/

---

## ✨ Features

- 🔐 User Authentication — Sign up, login & protected routes
- ✍️ CRUD Operations — Create, edit, and delete blog posts
- 🏷️ Tags & Categories — Filter posts by topic
- 🔍 Search — Find posts instantly by keyword
- 📄 Pagination — Clean browsing through all posts
